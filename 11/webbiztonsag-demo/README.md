# Webbiztonság alapjai - Demo

## Követelmények
- Python 3
- Chrome vagy Firefox

## Indítás (Windows)
1. Nyiss egy PowerShell-t a mappában.
2. Futtasd:
   ```
   python.exe server.py
   ```
3. Nyisd meg:
   - App:  http://localhost:8000
   - Evil: http://localhost:8001

## Indítás (Linux/macOS)
1. Nyiss egy Bash shell-t.
2. Futtasd:
  ```
  python server.py
  ```
