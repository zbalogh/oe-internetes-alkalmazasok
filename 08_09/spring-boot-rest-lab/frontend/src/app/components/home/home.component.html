Welcome to User Management System

This is a demonstration application built with Angular and Spring Boot. It showcases a complete CRUD (Create, Read, Update, Delete) implementation for managing user data.

Features

  • View all users in the system
  • Add new users with name and email
  • Update existing user information
  • Delete users from the system
  • RESTful API communication with Spring Boot backend

Getting Started

Click on the Users link in the navigation to view and manage users.
